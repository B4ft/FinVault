Why It Works

The AI loads recent transactions before processing your message:

// Loads tx descriptions (may contain
// injected instructions) — no sanitization
List<Transaction> recent =
  userService.getRecentTransactions(userId);

// Passes both to AI — indirect injection
chatService.processMessage(
  userMessage, recentTransactions);

Fix: Treat all external data as untrusted. Sanitize transaction descriptions before passing to LLM context. Use structured data formats instead of free text.
