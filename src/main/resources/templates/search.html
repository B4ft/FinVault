Vulnerable Code

The vulnerability is in main.js:

// REFLECTED XSS
document.getElementById('search-term')
  .innerHTML = query; // ← unsafe!

// STORED XSS
tr.innerHTML = `<td>${tx.description}</td>`;
// ↑ description from DB, unsanitized

Fix: use textContent instead of innerHTML, or sanitize with DOMPurify.
